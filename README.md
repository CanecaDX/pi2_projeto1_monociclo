# CanecaDX-pi2_projeto1_monociclo
Repositório para o projeto um (1), da cadeira de Projeto Integrador II do curso de Engenharia de computação UNIPAMPA campus Bagé

para compilar no windows: 
gcc -Wall -Wextra -Wpedantic -std=c11 -I .\include .\mars_main.c .\src\*.c -o .\mars_c.exe

para testar no windows:
./mars_c.exe

Para compilar no linux:
gcc mars_main.c src/*.c -Iinclude -Wall -Wextra -o mars_main_c

para testar no linux:
./main_c
